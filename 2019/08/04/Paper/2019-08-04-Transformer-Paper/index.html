Abstract

дёҖдёӘе®Ңе…ЁеҹәдәҺ Attention зҡ„жһ¶жһ„гҖӮжӣҙе®№жҳ“е№¶иЎҢи®ӯз»ғпјҢжҸҗй«ҳи®ӯз»ғйҖҹеәҰгҖӮ

Introduction

RNN зҡ„еӣәжңүеұһжҖ§дҪҝе…¶йҡҫд»Ҙе№¶иЎҢеҢ–пјҲеҚідҪҝйҖҡиҝҮ factorization tricks е’Ң conditional computation еҸҜд»Ҙеҫ—еҲ°ж”№е–„пјүпјҢAttention еҜ№дҫқиө–е…ізі»е»әжЁЎпјҢдёҚиҖғиҷ‘иҫ“е…Ҙиҫ“еҮәзҡ„и·қзҰ»гҖӮжң¬ж–ҮжҸҗеҮәзҡ„ Transformer йҮҮз”ЁдәҶе®Ңе…Ёзҡ„ Attention жңәеҲ¶жҸҸиҝ°иҫ“е…Ҙе’Ңиҫ“еҮәзҡ„ж•ҙдҪ“дҫқиө–е…ізі»пјҢеңЁи®ӯз»ғйҖҹеәҰе’Ңж•ҲжһңдёҠйғҪжңүжҳҺжҳҫжҸҗеҚҮгҖӮ

Background

ByteNet е’Ң ConvS2S йҖҡиҝҮеҚ·з§ҜзҪ‘з»ңе№¶иЎҢи®Ўз®—жүҖжңүиҫ“е…Ҙе’Ңиҫ“еҮәдҪҚзҪ®зҡ„йҡҗеұӮиЎЁзӨәпјҢе®ғд»¬з”ЁжқҘе…іиҒ”д»Һд»»ж„ҸдёӨдёӘиҫ“е…ҘжҲ–иҫ“еҮәдҪҚзҪ®зҡ„дҝЎеҸ·зҡ„ж“ҚдҪңж•°йҡҸзқҖи·қзҰ»зҡ„еўһеҠ иҖҢеўһй•ҝпјҲByteNet еҜ№ж•°еўһй•ҝпјҢConvS2S зәҝжҖ§еўһй•ҝпјүпјҢеҜјиҮҙжӣҙеҠ йҡҫд»ҘеӯҰд№ еҲ°иҝңи·қзҰ»зҡ„дҫқиө–е…ізі»гҖӮTransformer е°ҶиҝҷдёӘж“ҚдҪңж•°еҮҸеҲ°еёёж•°пјҢиҷҪз„¶з”ұдәҺе№іеқҮжіЁж„ҸеҠӣеҠ жқғдҪҚзҪ®иҖҢеҜјиҮҙжңүж•ҲжҖ§йҷҚдҪҺпјҢдҪҶеҸҜд»ҘйҖҡиҝҮ Multi-Head Attention жқҘж¶ҲйҷӨиҝҷз§ҚеҪұе“ҚгҖӮ

Self-attentionпјҢд№ҹз§° intra-attentionпјҢе®ғйҖҡиҝҮе…іиҒ”еәҸеҲ—зҡ„дёҚеҗҢдҪҚзҪ®и®Ўз®—еәҸеҲ—зҡ„иЎЁзӨәгҖӮ

Model Architecture

Encoder and Decoder Stacks

  • Encoder
    • 6 еұӮпјҢжҜҸеұӮдёӨдёӘеӯҗеұӮпјҡдёҖдёӘ multi-head self-attention еұӮе’ҢдёҖдёӘдҪҚзҪ®ж•Ҹж„ҹзҡ„е…ЁиҝһжҺҘеұӮ
    • еӯҗеұӮе‘ЁеӣҙдҪҝз”Ёж®Ӣе·®иҝһжҺҘпјҢз„¶еҗҺж ҮеҮҶеҢ–пјҢеҚіжҜҸдёӘеӯҗеұӮзҡ„иҫ“еҮәдёәпјҡLayerNorm(x + Sublayer(x))
    • жүҖжңүеӯҗеұӮе’Ң embedding зҡ„иҫ“еҮәеқҮдёә 512 з»ҙ
  • Decoder
    • йҷӨдәҶ Encoder дёӯзҡ„еӯҗеұӮпјҢиҝҳеј•е…Ҙз¬¬дёүдёӘеӯҗеұӮпјҢеҚі encoder stack зҡ„иҫ“еҮә
    • дҝ®ж”№ self-attention еӯҗеұӮйҳІжӯўдҪҚзҪ®еҮәзҺ°еңЁеҗҺз»ӯдҪҚзҪ®пјҢиҝҷдёӘ mask дҝқиҜҒдҪҚзҪ® i зҡ„йў„жөӢеҖјеҸӘдҫқйқ иҫ“еҮәдёӯ i д№ӢеүҚзҡ„дҪҚзҪ®

Attention

дёҖдёӘ Attention еҮҪж•°пјҡе°ҶдёҖдёӘ query е’ҢдёҖз»„ key-value еҜ№жҳ е°„дёәдёҖдёӘ outputгҖӮqueryпјҢkeyпјҢvalueпјҢoutput йғҪжҳҜеҗ‘йҮҸпјҢoutput е°ұжҳҜжүҖжңү value зҡ„еҠ жқғжұӮе’ҢпјҢжқғйҮҚз”ұ query е’Ңзӣёеә”зҡ„ key зҡ„еҮҪж•°и®Ўз®—гҖӮ

Scaled Dot-Product Attention

В AttentionВ (Q,K,V)=softmaxвҒЎ(QKTdk)V\text { Attention }(Q, K, V)=\operatorname{softmax}\left(\frac{Q K^{T}}{\sqrt{d_{k}}}\right) V В AttentionВ (Q,K,V)=softmax(dkвҖӢвҖӢQKTвҖӢ)V

query е’Ң key зҡ„з»ҙеәҰдёә dkпјҢдёҖз»„ Q зҡ„з»ҙеәҰдёә nГ—dk

дёӨдёӘжңҖеёёз”Ёзҡ„ attention еҮҪж•°пјҡ

  • additive attentionпјҡдҪҝз”Ёеҗ«жңүдёҖдёӘйҡҗеұӮзҡ„еүҚйҰҲзҪ‘з»ңи®Ўз®—
  • dot-product attentionпјҡе®һйҷ…жӣҙеҝ«жӣҙзңҒз©әй—ҙ

dk иҫғе°Ҹж—¶дёӨз§Қж–№жі•е·®дёҚеӨҡпјҢdk жҜ”иҫғеӨ§ж—¶пјҢadditive жҜ” dot-product иЎЁзҺ°жӣҙеҘҪпјҢдҪңиҖ…д»–д»¬зҢңжөӢеӣ дёәеўһй•ҝеӨӘеӨ§еҜјиҮҙ softmax еҫҲе°Ҹзҡ„жўҜеәҰпјҢжүҖд»Ҙз”Ё sqrt(dk) scale дәҶгҖӮ

Multi-Head Attention

В MultiHead(Q,K,V)=В ConcatВ (head,В ,вҖҰ,В headВ h)WO\text { MultiHead(} Q, K, V )=\text { Concat (head, }, \ldots, \text { head }_{\mathrm{h}} ) W^{O} В MultiHead(Q,K,V)=В ConcatВ (head,В ,вҖҰ,В headВ hвҖӢ)WO

В whereВ headВ i=В AttentionВ (QWiQ,KWiK,VWiV)\text { where head }_{i}=\text { Attention }\left(Q W_{i}^{Q}, K W_{i}^{K}, V W_{i}^{V}\right) В whereВ headВ iвҖӢ=В AttentionВ (QWiQвҖӢ,KWiKвҖӢ,VWiVвҖӢ)

WiQвҲҲRdmodelГ—dk,WiKвҲҲRdmodelГ—dk,WiVвҲҲRdmodelГ—dvW_{i}^{Q} \in \mathbb{R}^{d_{\mathrm{model}} \times d_{k}}, W_{i}^{K} \in \mathbb{R}^{d_{\mathrm{model}} \times d_{k}}, W_{i}^{V} \in \mathbb{R}^{d_{\mathrm{model}} \times d_{v}} WiQвҖӢвҲҲRdmodelвҖӢГ—dkвҖӢ,WiKвҖӢвҲҲRdmodelвҖӢГ—dkвҖӢ,WiVвҖӢвҲҲRdmodelвҖӢГ—dvвҖӢ

WOвҲҲRhdvГ—dmodelW^{O} \in \mathbb{R}^{h d_{v} \times d_{\mathrm{model}}} WOвҲҲRhdvвҖӢГ—dmodelвҖӢ

  • dk = dv = dmodel/h = 64
